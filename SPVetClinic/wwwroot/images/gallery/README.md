# Galería de Fotos - San Pablo Vet Clínic

Este directorio contiene las fotos proporcionadas por el equipo de publicidad.

## Estructura de Carpetas

```
gallery/
├── hero/           # Fotos para el carrusel del Hero (mínimo 3-4 fotos)
├── clinic/         # Fotos de las instalaciones de la clínica
├── team/           # Fotos del equipo médico y staff
├── services/       # Fotos de servicios siendo realizados
├── patients/       # Fotos de pacientes (con permiso de dueños)
└── equipment/      # Fotos de equipos e instrumentos
```

## Especificaciones de Imágenes

### Hero Carousel
- **Tamaño recomendado:** 1920x1080 px (o proporcional 16:9)
- **Formato:** JPG o WebP (preferido para mejor compresión)
- **Peso máximo:** 500KB por imagen
- **Cantidad:** 3-5 imágenes

### Fotos de Equipo/Especialistas
- **Tamaño recomendado:** 400x400 px (cuadrado)
- **Formato:** JPG o WebP
- **Peso máximo:** 100KB por imagen

### Fotos de Servicios/Instalaciones
- **Tamaño recomendado:** 800x600 px
- **Formato:** JPG o WebP
- **Peso máximo:** 200KB por imagen

## Nomenclatura Sugerida

- `hero-1.webp`, `hero-2.webp`, etc.
- `team-dr-vargas.webp`, `team-dra-munoz.webp`
- `clinic-recepcion.webp`, `clinic-quirofano.webp`
- `service-ecografia.webp`, `service-laboratorio.webp`

## Optimización

Antes de subir, optimizar imágenes con:
- [Squoosh](https://squoosh.app/) - Herramienta de Google
- [TinyPNG](https://tinypng.com/) - Compresión online
